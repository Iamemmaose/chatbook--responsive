Chatbook Project

Chatbook is a social media web interface inspired by popular platforms like Facebook. It allows users to view stories, post updates, check notifications, and interact with friends. The project focuses on HTML, CSS, and JavaScript with a responsive design and a dark mode toggle feature.



Features

Responsive Layout: Works on desktops, tablets, and mobile devices.

Header Navigation: Logo, notifications, messages, and profile menu.

Search Functionality: Search bar in the header.

Profile & Settings Menu: Toggleable menu with user details, settings, and dark mode switch.

Dark Mode: Clickable button to switch between light and dark themes. Theme preference is saved in localStorage.

Left Sidebar: Quick links and shortcuts to groups, friends, and pages.



Main Content:

Stories gallery

Post creation area with status visibility options

Posts feed with likes, comments, and share actions

Right Sidebar: Events, advertisements, and online friends list.

Footer: Simple copyright notice.

Technologies Used

HTML5 – Structure and content

CSS3 – Styling, responsive layout, and dark mode

JavaScript – Interactivity (settings menu, dark mode toggle, localStorage)

Font Awesome – Icons for UI elements


How to Use

Open index.html in a browser.

Click the profile image to toggle the settings menu.

Use the dark mode button in the settings menu to switch themes.

Scroll through posts, stories, and sidebar content.


Notes

All images are placeholders; replace with your own content if needed.

Theme preference persists across sessions using localStorage.
